## 소개

페이스북 인앱 브라우저에서는 기존에 사용하던 브라우저와의 연동이 되지 않아 많은 불편함을 겪습니다.
특히나 WebRTC 등 몇몇 기능의 이용이 불가능하여 사용자가 좋지 않는 경험을 할 수 있기에 미리 확인하여
유저에게 페이스북에서 나가서 브라우저를 켜서 들어올 것을 권장하기 위해 이 라이브러리를 만들었습니다.

## 설치

```bash
npm i detect-facebook-in-app-browser # npm

yarn add detect-facebook-in-app-browser # yarn
```

## 사용법

```ts
import isFacebookInAppBrowser from 'detect-facebook-in-app-browser';

if (isFacebookInAppBrowser()) {
  // do something...
}
```
